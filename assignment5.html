Extra Challenge: Explore

Do you see the hand wave emoji on the first line of the above contact form? 👋 I found a full list of emojis which are included as part of charset UTF-8. This adds a whim of fun to the text and can be read by all browsers using UTF-8 even though it appears as a small image.

Another attribute I added for this assignment is the pattern attribute to validate the email address in the form. There was conflicting advice on StackOverflow regarding the best RegEx to use for this, and I didn't take the time to review the RegEx in detail. I chose this regex:
< input type="email" pattern="[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,4}$" />
Even though this was an interesting find, the authors also noted that type="email" will basically do the validation for you even without using the pattern attribute.

Did you notice the change in font for the Regular Expression I printed above? I found the HTML code element, shown as the opening tag <code> and closing tag </code> will format this text without having to set a custom style.

Finally, I discovered how to escape HTML characters so the browser can print them without interpreting. I used the text representation of a left bracket instead of typing '<', since the browser has a specialized use for brackets.
